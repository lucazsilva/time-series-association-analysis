# Time Series Association Analysis

This project investigates associations between different time series using exploratory analysis, preprocessing techniques, correlation analysis, lag analysis, and visualization.

## Objectives

- Organize and clean time series datasets
- Explore temporal patterns
- Identify associations between variables
- Test lagged relationships
- Produce clear visualizations and reproducible outputs

## Project structure

- `data/raw`: original datasets
- `data/processed`: cleaned datasets
- `notebooks`: step-by-step analyses
- `scripts`: reusable functions
- `outputs`: figures and tables
