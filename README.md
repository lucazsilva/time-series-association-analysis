# Time Series Association Analysis

This project investigates associations between different time series using R.

## Objectives

- Organize and clean time series datasets
- Explore temporal patterns
- Preprocess time series data
- Test correlations and lagged associations
- Produce reproducible figures and tables

## Tools

- R
- tidyverse
- lubridate
- ggplot2
- forecast
- tseries
- zoo
- tsibble
- feasts

## Project structure

- `data/raw`: original data
- `data/processed`: cleaned data
- `scripts`: R scripts for each analysis step
- `outputs/figures`: exported plots
- `outputs/tables`: exported tables
- `reports`: final report in Quarto
